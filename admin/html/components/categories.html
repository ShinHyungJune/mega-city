    1

    하위 목차 등록

    왼쪽 목차 영역에서 목차명 위에서 마우스 오른쪽 클릭하시면 4개의 메뉴가 나옵니다.
    ‘하위 목차등록’ 을 선택하시면 오른쪽 목차 편집 영역에서 목차명 입력 후 ‘등록’버튼을 누리시면 생성됩니다.

    2

    하위 게시글 등록

    게시글을 등록하고 싶은 목차명 위에서 마우스 오른쪽 클릭하신 후 ‘하위 게시글등록’ 을 선택하시면
    오른쪽 목차 편집 영역에서 게시글의 내용을 등록하실 수 있습니다.

    3

    이름변경

    변경하고 싶은 목차 위에서 마우스 오른쪽 클릭하신 후 ‘이름변경’을 선택하시고
    오른쪽 목차 편집 영역에서 목차명을 수정한 후 ‘수정’버튼을 누르시면 변경됩니다.

    4

    삭제

    변경하고 싶은 목차 위에서 마우스 오른쪽 클릭하신 후 ‘이름변경’을 선택하시고
    오른쪽 목차 편집 영역에서 목차명을 수정한 후 ‘수정’버튼을 누르시면 변경됩니다.
