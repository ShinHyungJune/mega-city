Won is the currency of South Korea.
Between 1902 and 1910, the currency was first circulated under the name "Won," and it appeared in the Korean currency from 1945 to 1953.
The current won was introduced in 1962.
The Chinese character "원" was used before the 1953 currency reform, and now it is written only in Korean without Chinese characters to distinguish it from the old circle.
There used to be a "before" in the lower part of the won, which is now only used for accounting purposes.
The ISO 4217 code is KRW and the symbol is ₩. The English name is WON.
